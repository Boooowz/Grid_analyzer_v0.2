#
ΑΠΑΙΤΟΥΜΕΝΗ ΜΟΡΦΗ ΑΡΧΕΙΟΥ
==========================
Το αρχείο πρέπει να έχει στην πρώτη γραμμή τουλάχιστον τις στήλες:
ΑΦΜ, Όνομα, Επώνυμο, Ημ/νια, Απασχόληση

Η στήλη "Διάλειμμα" αξιοποιείται όταν υπάρχει. Οι υπόλοιπες στήλες μπορούν να υπάρχουν κανονικά.

ΚΑΝΟΝΕΣ ΥΠΟΛΟΓΙΣΜΟΥ
===================
- Ημέρες εργασίας: πόσες διαφορετικές ημερομηνίες έχουν αναγνωρισμένο ωράριο εργασίας.
- Ώρες εργασίας: διάρκεια του δηλωμένου ωραρίου χωρίς αφαίρεση διαλείμματος, π.χ. 10:00-18:00 = 8 ώρες, ακόμη κι αν υπάρχει "Εντός 20".
- Νυχτερινές ώρες: ώρες που πέφτουν στο διάστημα 22:00-06:00.
- Κυριακή: ημερολογιακή Κυριακή από 00:00 έως 24:00.
- Βάρδιες που περνούν τα μεσάνυχτα σπάνε χρονικά.
  Παράδειγμα: Σάββατο 21:00-01:00 δίνει 1 ώρα Κυριακής, δηλαδή 00:00-01:00.
- Νυχτερινές ώρες Κυριακής: ώρες που είναι ταυτόχρονα Κυριακή και νυχτερινές.
- Ημέρες άδειας: οποιαδήποτε γραμμή στο πεδίο «Απασχόληση» περιέχει ένδειξη άδειας, π.χ. «Κανονική άδεια», εξαιρείται πλήρως από ημέρες και ώρες εργασίας, ακόμη κι αν υπάρχει τυπικά ωράριο στην ίδια γραμμή.
- ΜΗ ΕΡΓΑΣΙΑ και ΑΝΑΠΑΥΣΗ/ΡΕΠΟ: εξαιρούνται από τους υπολογισμούς.
- Διάλειμμα: δεν αφαιρείται από τις ώρες. Θεωρείται ότι περιλαμβάνεται στο δηλωμένο ωράριο και εμφανίζεται μόνο ενημερωτικά.

ΣΗΜΑΝΤΙΚΟ
=========
Το πρόγραμμα υπολογίζει μόνο ημέρες και ώρες. Δεν υπολογίζει μισθοδοσία, προσαυξήσεις, εισφορές ή τελικές αποδοχές.
