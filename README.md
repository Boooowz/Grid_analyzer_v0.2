#
ΑΠΑΙΤΟΥΜΕΝΗ ΜΟΡΦΗ ΑΡΧΕΙΟΥ
==========================
Το αρχείο πρέπει να έχει στην πρώτη γραμμή τουλάχιστον τις στήλες:
ΑΦΜ, Όνομα, Επώνυμο, Ημ/νια, Απασχόληση

Η στήλη "Διάλειμμα" αξιοποιείται όταν υπάρχει. Οι υπόλοιπες στήλες μπορούν να υπάρχουν κανονικά.

ΚΑΝΟΝΕΣ ΥΠΟΛΟΓΙΣΜΟΥ
===================
- Ημέρες εργασίας: πόσες διαφορετικές ημερομηνίες έχουν αναγνωρισμένο ωράριο εργασίας.
- Ώρες εργασίας: διάρκεια του δηλωμένου ωραρίου χωρίς αφαίρεση διαλείμματος, π.χ. 10:00-18:00 = 8 ώρες, ακόμη κι αν υπάρχει "Εντός 20".
- Νυχτερινές ώρες: ώρες που πέφτουν στο διάστημα 22:00-06:00.
- Κυριακή: ημερολογιακή Κυριακή από 00:00 έως 24:00.
- Βάρδιες που περνούν τα μεσάνυχτα σπάνε χρονικά.
  Παράδειγμα: Σάββατο 21:00-01:00 δίνει 1 ώρα Κυριακής, δηλαδή 00:00-01:00.
- Νυχτερινές ώρες Κυριακής: ώρες που είναι ταυτόχρονα Κυριακή και νυχτερινές.
- Αργίες: οι επιλεγμένες αργίες μετριούνται ακριβώς όπως η Κυριακή (00:00-24:00) και προστίθενται στις ίδιες στήλες με την Κυριακή. Π.χ. εργασία στις 25 Μαρτίου, οποιαδήποτε ημέρα της εβδομάδας κι αν είναι, καταγράφεται στις ώρες Κυριακής.
- Ημέρες άδειας: οποιαδήποτε γραμμή στο πεδίο «Απασχόληση» περιέχει ένδειξη άδειας, π.χ. «Κανονική άδεια», εξαιρείται πλήρως από ημέρες και ώρες εργασίας, ακόμη κι αν υπάρχει τυπικά ωράριο στην ίδια γραμμή.
- ΜΗ ΕΡΓΑΣΙΑ και ΑΝΑΠΑΥΣΗ/ΡΕΠΟ: εξαιρούνται από τους υπολογισμούς.
- Διάλειμμα: δεν αφαιρείται από τις ώρες. Θεωρείται ότι περιλαμβάνεται στο δηλωμένο ωράριο και εμφανίζεται μόνο ενημερωτικά.

ΕΠΙΛΟΓΗ ΜΗΝΑ ΚΑΙ ΑΡΓΙΩΝ
=======================
Πάνω από το πεδίο φόρτωσης Excel υπάρχουν δύο νέα controls:

1. **Μήνας υπολογισμού** — επιλογή μήνα και έτους. Καθορίζει για ποιον μήνα/έτος προτείνονται οι ελληνικές δημόσιες αργίες. Δεν φιλτράρει τις γραμμές του Excel — όλα τα δεδομένα συνεχίζουν να επεξεργάζονται κανονικά.

2. **Αργίες μήνα** — αυτόματη πρόταση των ελληνικών δημόσιων αργιών που πέφτουν στον επιλεγμένο μήνα, με checkboxes ώστε να μπορούν να απενεργοποιηθούν:
   - Σταθερές: 1/1 Πρωτοχρονιά, 6/1 Θεοφάνεια, 25/3, 1/5 Πρωτομαγιά, 15/8 Δεκαπενταύγουστος, 28/10, 25/12 Χριστούγεννα, 26/12 Σύναξη Θεοτόκου.
   - Κινητές βάσει Ορθόδοξου Πάσχα: Καθαρά Δευτέρα, Μ. Παρασκευή, Δευτέρα Πάσχα, Αγίου Πνεύματος.

3. **Προσθήκη επιπλέον αργίας (έως 3)** — date pickers για χειροκίνητη εισαγωγή έως 3 επιπλέον ημερομηνιών εκτός των επίσημων αργιών.

Οι ώρες των αργιών συγχωνεύονται στις στήλες Κυριακής. Στις «Παρατηρήσεις» καταγράφεται ποιες ημερομηνίες μετρήθηκαν ως αργίες. Στις «Αναλυτικές Γραμμές» υπάρχει χωριστή πληροφοριακή στήλη «Αργίες που αφορούνται».

ΣΗΜΑΝΤΙΚΟ
=========
Το πρόγραμμα υπολογίζει μόνο ημέρες και ώρες. Δεν υπολογίζει μισθοδοσία, προσαυξήσεις, εισφορές ή τελικές αποδοχές.
